Switch 开关 ​

用于在两种状态之间切换，常见于启用 / 禁用、开 / 关、配置项切换等场景。

基础用法 ​

绑定 v-model 到一个布尔值即可使用。

<template>
  <div class="switch-demo-row">
    <moe-switch v-model="value1" />
    <moe-switch
      v-model="value2"
      style="--moe-switch-on-color: #13ce66; --moe-switch-off-color: #ff4949"
    />
  </div>
</template>

<script setup lang="ts">
import { ref } from 'vue'

const value1 = ref(true)
const value2 = ref(false)
</script>

<style scoped>
.switch-demo-row {
  display: flex;
  align-items: center;
  gap: 16px;
}
</style>

尺寸 ​

通过 size 控制 Switch 尺寸，支持 large、default 和 small。

<template>
  <div class="switch-demo-column">
    <moe-switch v-model="value" size="large" active-text="Open" inactive-text="Close" />
    <moe-switch v-model="value" active-text="Open" inactive-text="Close" />
    <moe-switch v-model="value" size="small" active-text="Open" inactive-text="Close" />
  </div>
</template>

<script setup lang="ts">
import { ref } from 'vue'

const value = ref(true)
</script>

<style scoped>
.switch-demo-column {
  display: grid;
  gap: 12px;
}
</style>

通过 active-text 和 inactive-text 展示开关状态文本；使用 inline-prompt 可以将文本显示在开关内部。

    <moe-switch v-model="value2" inline-prompt active-text="是" inactive-text="否" />
    <moe-switch v-model="value3" inline-prompt active-text="Y" inactive-text="N" />
  </div>
</template>

<script setup lang="ts">
import { ref } from 'vue'

const value1 = ref(true)
const value2 = ref(true)

</style>

阻止切换 ​

设置 before-change 可以在切换前执行校验或异步确认。返回 false 或 Promise reject 时会阻止切换。

<template>
  <div class="switch-demo-row">
    <moe-switch v-model="value1" :loading="loading1" :before-change="beforeChange1" />
    <moe-switch v-model="value2" :loading="loading2" :before-change="beforeChange2" />
  </div>
</template>

<script setup lang="ts">
import { ref } from 'vue'

import { MoeMessage } from '@moe-ui/components'

const value1 = ref(false)
const value2 = ref(false)
const loading1 = ref(false)
const loading2 = ref(false)

const beforeChange1 = () => {
  loading1.value = true
  return new Promise<boolean>((resolve) => {
    window.setTimeout(() => {
      loading1.value = false
      MoeMessage.success('切换成功')
      resolve(true)
    }, 1000)
  })
}

const beforeChange2 = () => {
  loading2.value = true
  return new Promise<boolean>((_, reject) => {
    window.setTimeout(() => {
      loading2.value = false
      MoeMessage.error('切换失败')
      reject(new Error('blocked'))
    }, 1000)
  })
}
</script>

<style scoped>
.switch-demo-row {
  display: flex;
  align-items: center;
  gap: 16px;
}
</style>

API ​

Attributes ​

名称类型默认值说明
model-value / v-modelboolean | string | numberfalse绑定值，应该等于 active-value 或 inactive-value
disabledbooleanfalse是否禁用
loadingbooleanfalse是否加载中
size'large' | 'default' | 'small''default'尺寸
widthstring | number—Switch 宽度
inline-promptbooleanfalse是否在开关内部显示文字或图标
active-iconstring | Component—开启状态图标，优先级高于 active-text
inactive-iconstring | Component—关闭状态图标，优先级高于 inactive-text
active-textstring''开启状态文字
inactive-textstring''关闭状态文字
active-valueboolean | string | numbertrue开启状态值
inactive-valueboolean | string | numberfalse关闭状态值
namestring''原生 name 属性
before-change() => boolean | Promise<boolean>—切换前钩子，返回 false 或 reject 时阻止切换
