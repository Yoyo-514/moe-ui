Tooltip 文字提示 ​

用于在鼠标悬停或点击时展示轻量提示信息。Tooltip 是后续 Popconfirm、Dropdown 等浮层组件的基础能力。

基础用法 ​

<template>
  <moe-tooltip content="这是一段 Tooltip 提示内容">
    <moe-button type="primary">悬停查看</moe-button>
  </moe-tooltip>
</template>

受控显示 ​

<template>
  <div class="tooltip-controlled-demo">
    <moe-tooltip v-model:visible="visible" content="受控 Tooltip" trigger="click">
      <moe-button>点击触发</moe-button>
    </moe-tooltip>
    <moe-button type="primary" @click="visible = !visible">
      {{ visible ? '隐藏' : '显示' }} Tooltip
    </moe-button>
  </div>
</template>

<script setup lang="ts">
import { ref } from 'vue'

const visible = ref(false)
</script>

<style scoped>

  display: flex;
  align-items: center;

}
</style>

Tooltip API ​

Props ​

名称类型默认值说明
contentstring''提示内容
placementPlacement'bottom'出现位置
triggerTooltipTrigger | TooltipTrigger[]'hover'触发方式
disabledbooleanfalse是否禁用
visible / v-model:visibleboolean—组件可见性
showAfternumber0延迟显示时间，单位 ms
hideAfternumber200延迟隐藏时间，单位 ms
offsetnumber12与触发元素的偏移距离
effect'dark' | 'light''dark'主题样式
transitionstring'fade'过渡动画名称
popper-optionsobject{}popper.js 参数，请参考 popper.js 文档

TooltipTrigger 类型：

ts
type TooltipTrigger = 'hover' | 'click'

Placement 与 Options 来自 @popperjs/core，支持 top、bottom-start、right-end 等 Popper 方位和 Popper 配置，更多参数可参考 popper.js 文档。

Emits ​

名称参数说明
show—Tooltip 显示时触发
hide—Tooltip 隐藏时触发

Slots ​

名称说明
default触发 Tooltip 的内容
content自定义 Tooltip 内容

Exposes ​

名称类型说明
show() => void显示 Tooltip
hide() => void隐藏 Tooltip
